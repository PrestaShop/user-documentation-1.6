# Vooraf geïnstalleerde modules gebruiken

Dit hoofdstuk verkent het configuratieproces van alle standaardmodules in PrestaShop 1.6

Er zijn meer dan 130 vooraf geïnstalleerde modules beschikbaar in 25 secties. Modulesecties worden alleen getoond als ze minimaal één module bevatten.

Sommige modules hebben configuratiepagina's, die u toegang geven tot en aantal tools en informatiepagina's.

Op elke configuratiepagina zijn algemene knoppen beschikbaar aan de bovenkant van het scherm:

* **Terug**. Een link om terug naar de modulelijst te gaan.
* **Vertalen**. Hiermee wordt u naar de pagina "Vertalingen" onder het menu "Lokalisatie" gebracht om de module te vertalen.
* **Op updates controleren**. Controleert of er een nieuwe versie van deze module beschikbaar is (in de repository van PrestaShop of Addons-winkel), en als deze er is wordt de module meteen bijgewerkt.
* **Beheer haken**. Dit is een snelkoppeling naar de pagina "Posities" onder het menu "Module en Services". Vanaf daar kunt u de locatie van de modules op de frontend wijzigen of de Live Edit tool openen. De pagina "Posities" is zodanig geconfigureerd dat alleen de haken van de huidige module worden getoond.

Sommige modules worden niet geïnstalleerd vanaf bestanden op uw server, maar worden gelijk vanaf de Addons-website gedownload. Hiermee bent u verzekerd van de meest recente versie van een module.

De "Must have" modules zijn ook beschikbaar vanaf de Addons-site, maar zijn niet gratis beschikbaar. Deze modules worden niet getoond in dit hoofdstuk.
